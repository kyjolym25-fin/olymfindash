# 올림피아드교육 손익실적 대시보드 — Google Apps Script 배포 가이드

## 📁 파일 구성

```
gas_project/
├── Code.gs           ← 서버사이드 로직 (사용자 인증, 데이터 저장, 집계)
├── index.html        ← 대시보드 UI (클라이언트 전체)
├── appsscript.json   ← GAS 프로젝트 설정
└── README.md         ← 이 파일
```

---

## 🚀 배포 방법 (처음 설정)

### 1단계 — 구글 스프레드시트 생성
1. [Google Sheets](https://sheets.google.com) 접속 → 새 스프레드시트 생성
2. 시트 이름을 아무거나 지어도 됩니다 (예: `올피 손익 DB`)
3. 스프레드시트 URL의 ID 복사  
   예) `https://docs.google.com/spreadsheets/d/`**`1aBcDeFgHiJk...`**`/edit`

### 2단계 — Apps Script 프로젝트 생성
1. 스프레드시트 메뉴 → **확장 프로그램 → Apps Script** 클릭
2. 프로젝트 이름을 `올피 손익 대시보드`로 변경

### 3단계 — 파일 업로드

#### Code.gs
1. 기존 `코드.gs` 내용을 전체 삭제
2. `Code.gs` 파일 전체 내용을 붙여넣기 → **저장(Ctrl+S)**

#### index.html
1. 왼쪽 `+` 버튼 → **HTML 파일** → 이름: `index`
2. `index.html` 파일 전체 내용을 붙여넣기 → **저장**

#### appsscript.json (프로젝트 설정)
1. Apps Script 편집기 → 왼쪽 **프로젝트 설정(⚙️)** → "appsscript.json 파일 표시" 체크
2. 다시 편집기로 돌아오면 `appsscript.json` 파일 탭이 생김
3. 내용 전체 교체 후 저장

### 4단계 — 웹앱으로 배포
1. 오른쪽 상단 **배포 → 새 배포** 클릭
2. 유형 선택 → **웹 앱**
3. 설정:
   - 설명: `v1.0`
   - 다음 사용자로 실행: **나(관리자 계정)**
   - 액세스 권한: **모든 사용자** (로그인 없이 URL 접근 가능)
4. **배포** 클릭
5. 생성된 **웹 앱 URL** 복사 → 직원들에게 공유

> ⚠️ 처음 배포 시 구글 계정 권한 요청 팝업이 나타납니다.  
> **고급 → (앱 이름)으로 이동 → 허용**을 클릭하세요.

## 🔄 매월 데이터 업데이트

1. 대시보드 접속 → 로그인 (관리자 계정)
2. **⚙️ 관리** 탭 → **📤 손익계산서 로우 업로드**
3. 해당 월 손익계산서 엑셀 파일 선택 (`.xlsx`)
   - 파일에 **로우** 시트가 있어야 합니다
   - 컬럼: `매출/판매관리비 구분 | 원장계정과목 | C/C | 월 | 본부 구분 | 캠퍼스 구분 | 과목 구분 | 학년 구분 | 금액`
4. 업로드 완료 → 전체 대시보드 자동 갱신

재원생 데이터는 **📅 재원생 데이터 직접 입력** 섹션에서 수동 입력합니다.

## ❓ 자주 묻는 문제

**Q. 배포 후 흰 화면만 나와요**  
→ Apps Script 편집기 → **실행 로그** 확인. `doGet` 함수 오류 여부 확인.

**Q. 데이터가 저장되지 않아요**  
→ 배포 설정에서 "다음 사용자로 실행"이 **나**로 설정되어 있는지 확인.

**Q. 업로드 후 데이터가 반영 안 돼요**  
→ 스프레드시트에 `RAW_DATA` 시트가 생성됐는지 확인. 없으면 첫 업로드 시 자동 생성됩니다.

**Q. 직원들이 URL에 접근 못해요**  
→ 배포 설정 → 액세스 권한이 **모든 사용자**인지 확인.
